About stegosaurus

Many of the sharing platforms used every day compress images to keep download speeds high and overhead low. Sharing an image that has been encrypted using the stegosaurus service over one of these platforms could result in the loss of the encrypted content. Platforms that will compromise the data embedded in your stegosaurus image file include: Facebook, Twitter, Instagram, even standard text message. Email, iMessage, and AirDrop are services safe to use when sharing stegosaurus images. Images will be hosted on the stegosaurus server for 24 hours and will be accessible via link. This link can be used to continue sharing over platforms that compress images.
